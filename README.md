# Infinity Group - Luxury Real Estate

Welcome to the official web platform for **Infinity Group**, a premium builder and developer specializing in luxurious living spaces, boutique workspaces, and integrated townships.

## Overview
This repository contains the front-end code for the Infinity Group website. The site is designed with a modern, sophisticated aesthetic tailored for the high-end real estate market. It features:
- A responsive, glassmorphism-inspired UI
- Dynamic project portfolio filtering (Commercial, Residential, Township)
- Smooth scrolling and interactive elements
- Integrated zero-backend lead generation forms (connected directly to Google Sheets)

## Technologies Used
- HTML5 / CSS3
- Vanilla JavaScript
- Tailwind CSS (Utility-first styling approach via custom CSS implementation)

## Project Structure
- `index.html` - The Home Page featuring the hero section, company overview, and featured properties.
- `projects.html` - The unified portfolio grid with category filters.
- `about.html` - Information regarding the company's legacy, vision, and leadership.
- `contact.html` - Direct contact details and the primary enquiry form.
- `faq.html`, `privacy-policy.html`, `terms-and-conditions.html` - Legal and informational pages.
- `main.js` - Global JavaScript handling mobile menus, animations, scroll effects, and aggressive lead generation popups.
- `asset/` - Directory containing high-resolution renders and project images.
- `projects/` - Directory containing individual, detailed pages for each real estate project.

## Lead Generation
The website is equipped with aggressive lead generation mechanisms:
1. **Floating Action Button:** A persistent WhatsApp button that pre-fills user inquiries.
2. **Pop-up Form:** A time-triggered modal that captures user details and routes them seamlessly.
3. **Google Sheets Integration:** Forms are fully wired to Google Apps Script for instant lead logging without the need for traditional server infrastructure.
